# ElderGuard
**ElderGuard** is a smart **critical injury detection system** for elderly safety. Using **ESP32**, it monitors **motion (PIR), distress sounds (MAX9814), heart rate &amp; SpO2 (MAX30102), and environment (DHT11)**. It sends **real-time alerts** to caregivers via IoT.
